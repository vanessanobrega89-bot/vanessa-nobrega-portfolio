# Vanessa Nobrega — Communication d'innovation

Landing page personnelle, version salariat.

## Stack

- HTML autoporteur (1 fichier)
- CSS inline
- Photo en base64
- Google Fonts via CDN (DM Sans, Inter, JetBrains Mono)
- Pas de build, pas de framework, pas de bundler

## Déploiement

Hébergé sur [Vercel](https://vercel.com). Le déploiement est automatique à chaque push sur la branche `main`.

L'URL `/` redirige vers `VNobrega-Landing-Salariat.html` via la configuration dans `vercel.json`.
